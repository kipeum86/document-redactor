# Safer offline DOCX redaction, clearer warnings, and reusable local policies

더 안전한 오프라인 DOCX 마스킹, 명확한 경고, 반복 작업용 로컬 정책

This release makes offline DOCX redaction safer, clearer, and easier to repeat.

이번 업데이트는 오프라인 DOCX 마스킹 작업을 더 안전하고, 더 알기 쉽게, 반복 사용하기 쉽게 개선했습니다.

## What Improved

- **More reliable redaction checks**
  The app is better at catching cases where text looks selected in preview but could remain in the final DOCX.

  **더 믿을 수 있는 마스킹 확인**
  미리보기에서는 선택된 것처럼 보였지만 최종 DOCX에 남을 수 있는 경우를 더 잘 잡도록 개선했습니다.

- **Clearer warnings for risky output**
  If a file still has remaining sensitive text, the download flow is more clearly separated from a clean, verified file. Risky override files are marked as `UNVERIFIED`.

  **위험한 결과물 표시 강화**
  민감한 문자열이 남아 있을 가능성이 있는 파일은 정상 검증 완료 파일과 더 명확히 구분됩니다. 이런 파일은 `UNVERIFIED` 표시가 붙습니다.

- **Reusable local redaction policies**
  You can export frequently used manual redaction terms and import them again for another document. This stays local; no account or server is involved.

  **반복 작업용 로컬 정책 파일 지원**
  자주 마스킹하는 이름, 회사명, 프로젝트명 등을 로컬 파일로 저장했다가 다른 문서에서 다시 불러올 수 있습니다. 계정이나 서버는 필요 없습니다.

- **Better handling of unsupported files**
  The app now rejects invalid DOCX files, macro-enabled files, and encrypted/password-protected packages earlier with clearer errors.

  **지원하지 않는 파일을 더 명확히 안내**
  잘못된 DOCX, 매크로가 포함된 파일, 암호화/비밀번호 보호 파일은 더 이른 단계에서 명확한 오류로 안내합니다.

- **Improved review counts and candidate details**
  Candidate totals and review lists are now more consistent, so it is easier to understand what needs to be checked before applying redaction.

  **검토 대상 숫자와 표시 개선**
  후보 개수와 목록 표시가 더 일관되게 맞춰져, 적용 전에 무엇을 확인해야 하는지 더 쉽게 볼 수 있습니다.

- **Still a single offline HTML file**
  The tool remains a single local HTML file with zero network requests. The current checked build is about **281 KB**.

  **여전히 HTML 파일 하나로 동작**
  도구는 계속 HTML 파일 하나로 실행되며, 네트워크 요청은 없습니다. 현재 확인된 빌드 크기는 약 **281 KB**입니다.

## Recommended Use

Download both files from the release:

- `document-redactor.html`
- `document-redactor.html.sha256`

Before opening the HTML file, verify it with the SHA-256 sidecar when possible.

## 권장 사용법

릴리즈에서 두 파일을 함께 받으세요:

- `document-redactor.html`
- `document-redactor.html.sha256`

가능하면 HTML 파일을 열기 전에 SHA-256 파일로 무결성을 확인해 주세요.
